# Steps to create backend

1.Generate package.json
   
    npm init -y
   
2.Create HTTP server

   a.Install & import "express" module
    
        npm install express
        
   b.Import express module
    

HTTP Req types (CRUD operation)

   GET  -- Read resources
    
   POST -- Craete a new resource

   PUT  -- Update a resource
    
   DELETE -- Delete a resource
    

USER API:

    GET http://127.0.0.1:3000/path/users
    POST http://127.0.0.1:3000/path/user
    PUT http://127.0.0.1:3000/path/users/<id>
    DELETE http://127.0.0.1:3000/path/users/<id>
    
    http://localhost:3000
